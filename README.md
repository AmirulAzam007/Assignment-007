## 🧠 Keen-Keeper

## 📌 Project Overview
Keen-Keeper is a personal relationship tracking web application designed to help users manage and maintain meaningful connections. It allows users to log interactions, track communication history, and visualize engagement patterns in a simple and intuitive interface.

The goal of this project is to make it easier to stay consistent with friends and important contacts through structured reminders and analytics.

---

## 🛠️ Technologies Used

- **React.js / Next.js** – Build the UI and component-based structure  
- **React Router DOM / App Router (Next.js)** – Handle page navigation and routing  
- **Tailwind CSS + Component Library (DaisyUI / shadcn etc.)** – Styling and responsive design  
- **Recharts** – Data visualization and analytics charts  

---

## ✨ Key Features

### 1. 📇 Friend Management System
Users can add and manage friends with detailed profiles including contact history, tags, and relationship goals.

### 2. 📊 Interaction Timeline Tracking
Every interaction (call, text, or video) is recorded and displayed in a timeline format for better visibility and tracking.

### 3. 📈 Analytics Dashboard
Interactive charts visualize user engagement patterns, helping users understand communication frequency and consistency over time.
